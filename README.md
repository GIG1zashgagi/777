# Практическая работа №7: Отладка программы различными способами

**Выполнил:** Завадский Илья  
**Группа:** 3ИСИП-423    
**Среда разработки:** Microsoft Visual Studio 2022 / .NET 6.0+

##  Цель практической работы

Произвести процедуру отладки программного обеспечения встроенными средствами среды программирования **Microsoft Visual Studio**. Научиться использовать точки останова, пошаговое выполнение, окна наблюдения, стек вызовов и другие инструменты отладчика для поиска и устранения ошибок в коде.

##  Структура проекта

Проект представляет собой единое консольное приложение на C#, объединяющее три самостоятельные задачи:


### Содержание приложения:

| № | Название | Описание |
|---|----------|----------|
| 1 | **Числа Фибоначчи** | Вычисление и вывод последовательности чисел Фибоначчи |
| 2 | **Каталог Галактик** | Демонстрация ошибки выхода за границы массива и обработка null-значений |
| 3 | **Анализатор гласных букв** | Подсчёт гласных букв в тексте с намеренной ошибкой регистра |
